# Verdict / «Троица»

Verdict — экспериментальный интерпретатор языка «Троица» для символического вывода на троичной логике (`да`, `нет`, `нез`). Проект поддерживает факты, правила, причины вывода, XAI-отчёты, абдукцию, онтологический fallback, ревизию убеждений и автоматическое формирование понятий.

## Быстрый старт

```bash
python -m pip install -r requirements.txt
python verdict9.py
```

Запуск API:

```bash
uvicorn api:app --reload
```

## Минимальный пример `.tri`

```text
факт симптом(иван) = да причина факт.
правило болен(X) :- симптом(X) = да причина правило(симптом_болезни).
цель: болен(иван) = Статус причина Причина.
```

## Структура

- `verdict9.py` — основной интерпретатор и CLI-запуск встроенного demo.
- `cognitive_modules.py` — ревизия убеждений и MDL-формирование понятий.
- `api.py` — FastAPI endpoint `/infer` для вывода по переданной базе знаний.
- `docs.txt` — подробная спецификация языка и архитектуры.
- `tests/` — базовые regression-тесты запускаемости и ключевой логики.

## Статус

Проект находится на стадии исследовательского прототипа. API и язык стабилизируются; для расширения DSL рекомендуется добавлять тесты на каждый новый синтаксический сценарий.
